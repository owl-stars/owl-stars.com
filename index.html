Spriteowl

Photoshop extension

Free
Spriteowl is a Photoshop extension that takes over the boring part when working with sprite images and CSS.
  • Retrieves x, y, width and height info of all sprite elements for you
  • Exports retina ready PNG files and all required media queries of your Photoshop sprite file
  • Supports preprocessors LESS, SASS/SCSS & Stylus
Find out more

Cssowl

Mixin library

Free
Cssowl is a mixin library that work with the Photoshop extension Spriteowl to help you developing easier and faster.
  • LESS, SASS/SCSS & Stylus support
  • Open source
  • Easy to use
